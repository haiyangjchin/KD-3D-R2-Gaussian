# 误差分布分析报告

## 问题描述

知识蒸馏训练中观察到：RMSE在5000次迭代后上升，而SSIM持续改善。

### 关键指标变化

- **最佳RMSE迭代**: iteration_5000 (RMSE=0.012040)
- **最终迭代**: iteration_10000 (RMSE=0.012664)
- **RMSE增加**: +5.18%
- **MAE增加**: +3.17%
- **最大误差增加**: +4.44%
- **大误差比例增加**: -8.68%

## 原因分析

1. **RMSE对异常值敏感**: RMSE对较大误差进行平方惩罚，少数大误差会显著提高RMSE
2. **误差分布变化**: 后期迭代可能产生更多的大误差像素
3. **SSIM-RMSE权衡**: SSIM关注结构相似性，RMSE关注像素级精度
4. **可能的局部过拟合**: 模型在某些区域过度适应，产生较大偏差

## 建议

1. 检查误差分布图，识别大误差区域
2. 考虑早停策略，在RMSE开始上升时停止训练
3. 增加正则化，防止过拟合
4. 分析SSIM和RMSE的权衡，根据应用需求选择合适指标
